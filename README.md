# ❤️daqiao-cli

# ![Npm版本](https://img.shields.io/badge/daqiao_cli-v0.0.2-green)


# 📖简介
用于快速搭建前端项目的命令行工具

# 模板
# ✅ [生成Vue3+TS+Vite+ElementPlus+Pinia+Axios+Mock的模板](https://www.npmjs.com/package/daqiao-cli)

```bash
# 使用模板创建新项目
daqiao create 

# 执行项目文件夹名称并选择模板创建新项目
daqiao create [projectName]

# 更新脚手架到最新版本
daqiao update

# 查看脚手架版本
daqiao -v
daqiao --version

```
